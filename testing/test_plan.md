# План тестирования DevSecOps Reporter

### Цели тестирования
- Проверить корректность парсинга всех поддерживаемых форматов
- Убедиться в правильной генерации HTML-отчёта и графиков
- Оценить удобство использования и читаемость отчёта

### Среда тестирования
- ОС: Windows 11, Ubuntu 22.04
- Python 3.11–3.13
- Браузеры: Chrome, Firefox, Edge

### Типы тестирования
- Функциональное (ручное + автоматическое в GitHub Actions)
- Пользовательское (опрос 12 одногруппников)
